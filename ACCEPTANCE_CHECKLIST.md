# 點單頁功能驗收清單

## 全局卡片

- [ ] 卡片由正確觸發位置展開，三角指向來源。
- [ ] 標準卡寬度 25%；飲品設定卡約為大產品卡 2/3。
- [ ] 最大高度不超過購物車，中間內容可滾動。
- [ ] 標題與底部按鈕不隨內容滾動。
- [ ] 卡片開啟時背景不可操作，點空白位置無效。
- [ ] 取消／X 不保存；確認後主畫面更新。
- [ ] 重新載入後需要保存的設定仍存在。

## 快捷點單與整理

- [ ] 快捷模式點產品不彈卡，立即加入購物車。
- [ ] 可組合、飯底、醬料、飲品等未完成數量正確。
- [ ] 未整理完成不能結帳。
- [ ] 一鍵組合先預覽再確認，不靜默改價。
- [ ] 自由搭配 A/B/C 不重複分配產品數量。
- [ ] 指定飲品只加入指定套餐。

## 產品與飲品設定

- [ ] 後台權重、必選／可選、單選／多選正確反映。
- [ ] 其他調整預設收起，點擊才展開。
- [ ] 相同設定合併數量；不同設定分行。
- [ ] 合併項目可拆分修改一份。
- [ ] 無選飲品顯示「無調整」，不顯示正常甜／正常冰。

## 狀態列與線上接單

- [ ] 待處理使用通知圖標及數量徽章。
- [ ] 線上使用 Wi-Fi／連線圖標並顯示真實同步狀態。
- [ ] 新訂單未點立即處理前不載入購物車或改狀態。
- [ ] 新單通知逾時後進入待處理，不會遺失。
- [ ] 接單、暫停、提早收工只在儲存後同步磨飯 App。

## 實機

- [ ] iPhone／Android 手機瀏覽效果圖與文件正常。
- [ ] 目標橫屏 POS 觸控區足夠，右手操作無遮擋。
- [ ] 連續新增、修改、取消、確認 30 次無狀態錯亂。
- [ ] 離線、重新連線、重新整理後狀態一致。
