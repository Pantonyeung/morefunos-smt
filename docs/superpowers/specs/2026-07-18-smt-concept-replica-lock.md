# More Fun／磨飯 SMT｜概念圖完整復刻鎖

版本：V1.0 Replica Lock  
日期：2026-07-18

## 目標

第一階段不做重新設計、不做風格再詮釋；以使用者提供的 SMT-00 至 SMT-16 十張概念圖作為唯一視覺基準，先完成整套可操作 UI 的完整復刻，再逐畫面微調。

## 復刻優先順序

1. 整體框架與比例
2. 頂欄、底部導航、工作區分欄
3. 暖橙＋米白＋深色文字配色
4. 固定綠／黃／紅／藍狀態語義
5. 卡片、按鈕、Chip、Badge、Drawer、Modal、Toast
6. 圖片比例、商品卡密度、字級、間距、邊框與陰影
7. 報表、設備、打印、狀態中心與 Gate 畫面

## 硬性限制

- 全介面使用繁體中文。
- 主導航固定：點單｜訂單｜堂食｜售罄｜更多。
- 保留已鎖定商業規則、Required／Pool／Link Up、訂單時間、打印、付款、同步及離線邏輯。
- 不把概念圖中的明顯錯字、簡體字、假資料或不可操作示意直接帶入正式版本。
- 第一階段不加入額外主題延伸、品牌粒子、沉浸式動畫或重新設計。
- `main` 保持不動；所有復刻工作在獨立分支完成。

## 驗收標準

- 使用者第一眼能辨認畫面與概念圖屬同一套 UI。
- 框架、比例、配色、主要元件及資訊層級先達到接近復刻，再討論優化。
- 1024、1280、1920 橫屏均維持核心按鈕可見與至少 48px 觸控尺寸。
- 全介面繁體中文，沒有簡體字或只靠圖標理解的核心操作。
