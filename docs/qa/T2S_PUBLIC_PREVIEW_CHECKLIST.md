# T2S 公網預覽驗收清單

分支：`test/t2s-display-adaptation`

## 公網可以驗證

- 點單頁大圖、小圖、純文字模式均維持每行四張，最後一行不足四張時不得拉闊。
- 所有產品圖片能顯示完整內容，不裁切、不拉伸，不再只見「餐點圖片」。
- 購物車、分類列、頂欄及底欄不互相遮擋；畫面內容只在指定區域滾動。
- 快捷飲品展開後不超過畫面三分一，關閉後底部五項導航全部可按。
- 依次切換點餐、訂單、堂食、售罄、更多，再由更多返回點餐及訂單。
- 更多頁維持兩欄三行；六個入口逐一打開、操作、關閉，遮罩不得殘留。
- 顯示與操作設定可保存，重新載入後仍保留；日結、報表、備份及系統頁不可錯位。

## 必須留待安裝程式驗證

- 商米內置小票機實際出紙。
- 兩部網絡後廚機及兩部網絡標籤機實際出紙、重試、改送及補印。
- 安卓橋接回傳 `printed` 或清楚失敗原因。
- 斷網啟動、內置後備頁、舊商米瀏覽器兼容及覆蓋安裝。

公網按鈕有反應、打印工作成功排隊或自動測試通過，都不可當作實體出紙完成。
