# SMT Slice 1 Manual QA

## 繁體中文與閱讀性
- [ ] 所有主導航、按鈕、狀態、錯誤及空白提示為繁體中文。
- [ ] 沒有只靠圖標才能理解的核心操作。
- [ ] 1024×600、1280×800 橫屏下正文仍清楚，不需放大瀏覽器。
- [ ] 商品代碼、名稱、價格及售罄文字不互相遮蓋。
- [ ] 圖片失效時顯示文字替代，不出現破圖圖標。

## 直覺操作
- [ ] 點商品後立即看到卡片、購物車行及總額回饋。
- [ ] Required 未完成清楚顯示尚欠數量。
- [ ] 付款方式選中後同時看到勾選、文字及底色。
- [ ] 保存失敗不清空購物車。
- [ ] 稍後處理新單後角標仍保留。

## 動效
- [ ] 一般模式沒有持續閃爍或背景漂浮。
- [ ] Reduced Motion 開啟後仍有清楚文字狀態。
- [ ] 快速連按不造成重複正式訂單。
